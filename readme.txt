Description:
This project involves a Soccer Management Database designed using SQLite3, accessible through a custom command-line interface. 
The database manages information about leagues, seasons, teams, players, and matches, allowing users to perform various queries 
such as viewing team rosters, match fixtures, and player participation across different seasons and leagues.

Files Included:
- soccer_management.db : The SQLite3 database file.
- cli.py : Python script for the command-line interface.
- readme.txt : This README file containing the YouTube video link.
- FinalProject_Description.pdf : Description of database and it's ER model.
